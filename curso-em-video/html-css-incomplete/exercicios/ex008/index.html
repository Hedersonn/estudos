Itálico/ Ênfase

Aqui temos um termo em itálico usando a tag I (não semantica).

Texto marcado

Podemos criar um texto marcado com a tag MARK.

E no outro parágrafos temos outro texto marcado no final.

Texto grande e texto pequeno

Estamos criando um Texto Grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Temos também o texto com a tag U (não semântica)

Texto Sobrescrito

Para inserir coisas do tipo x20+3 usamos a tag SUP

Texto Subscrito

Para inserir coisas do tipo H2O usamos a tag SUB
